It makes me feel better about not being able to remember the lyrics because neither 'pocket' nor 'sunshine' is sung very often.

How did I do this?

I used a dictionary to write the code in Java to create a word frequency of our song lyrics. A dictionary seemed like the best data structure for this task because its use of a key-value pair made it easy to keep track of the number of times a word appeared in the song. Alternatively I could have used a Queue, LinkedList, Graph, Stack, Tree, or Array. None of those seemed like a good option, however. I could have created an object called word and given it a private int value that kept track of the number of times each word was used and then used a Linkedlist but a HashMap was easier to implement. I then printed the value: key pair on a separate document. I made sure to remove punctuation and typeCast everything to lower case. This means that it is a not counting the same word as different words because of something like a comma or a capitalized letter.
